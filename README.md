# Transkit-app Tap

## How do I install these formulae?

`brew install transkit-app/tap/<formula>`

Or `brew tap transkit-app/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "transkit-app/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
